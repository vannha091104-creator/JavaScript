# JavaScript Practice

## Overview

This repository contains my JavaScript practice exercises focusing on DOM manipulation, element selection, attributes, classes, and CSS manipulation.

## Exercises

### Exercise 1
**Target:** Selecting Elements, Getting Values, and Setting Values

- Practice selecting DOM elements
- Get element values and properties
- Update element values and text content

### Exercise 2
**Target:** Attributes, Classes, and CSS Manipulation

- Work with HTML attributes
- Add, remove, and toggle CSS classes
- Manipulate element styles using JavaScript

### Exercise 3
**Target:** Attributes, Classes, and CSS Manipulation

- Practice manipulating tables and forms
- Modify attributes and styles dynamically
- Handle common DOM operations

## Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
